Develop Notepad using Tkinter in Python

What is Tkinter?

Python has a plethora of libraries and these 4 stands out mainly when it comes to GUI. There are as follows:


  • Tkinter
  • Python QT
  • wxPython
  • Kivy


Among all of these, Tkinter is preferred by a LOT of learners and developers just because of how simple and easy it is.

Steps to create Tkinter App

1

Importing the module – tkinter

There is no need to add external library to use Tkinter into your Applicaton. It's built in python library we just have to import it.


from tkinter import *

2

Create the main window (container)

We know every Application has one main windows in which we add all elements and alligns it according to needs. To create a main window, tkinter offers a method called Tk().


root = tkinter.Tk() [where root is the name of the main window object]

3

Add any number of widgets to the main window

There are a number of widgets which you can put in your tkinter application.


E.g. Buttons, Entry, Frame, Label, Listbox, MenuButton

4

Apply the event Trigger on the widgets.

We just have to write and call python function to execute particular command.
